📋 BRIEF PARA DESARROLLADOR - MAQUETA HTML + TAILWINDCSS

🎯 CONTEXTO DEL PROYECTO
Cliente:

Institución: Institución Educativa N°11017 – Nicolas la Torre García (Chiclayo, Perú)
Organización: ONG Voices of Help
Áreas involucradas: Trabajo Social y Psicología

Objetivo del Proyecto:
Crear un sistema web para digitalizar el proceso de evaluación socioeconómica y psicológica de estudiantes y sus familias. Actualmente usan documentos Word que deben ser convertidos a formularios web interactivos.
Objetivo de esta Maqueta:
Crear un prototipo visual HTML estático con TailwindCSS para que el cliente pueda:

Ver cómo lucirá el sistema final
Validar la estructura de los formularios
Aprobar el diseño antes de invertir en desarrollo backend


🛠️ ESPECIFICACIONES TÉCNICAS
Stack Requerido:
✅ HTML5 puro (semántico)
✅ TailwindCSS vía CDN 
✅ JavaScript Vanilla (para interacciones básicas)
✅ NO usar frameworks (React, Vue, etc.)
✅ NO necesitas backend (todo estático)
CDN de TailwindCSS:
html<script src="https://cdn.tailwindcss.com"></script>
```

### **Requisitos de Diseño:**
- ✅ **Responsive** (móvil, tablet, desktop)
- ✅ **Profesional y limpio** (no infantil)
- ✅ **Colores:** Azul institucional como primario, gris para secundarios
- ✅ **Tipografía:** Inter, Roboto o similar (Google Fonts)
- ✅ **Iconos:** Font Awesome o Heroicons vía CDN
```

```
   - Tipo: text
   - Ancho: 1 columna
   - Requerido: Sí

3. **Sexo**
   - Tipo: radio buttons (M / F)
   - Ancho: 1 columna
   - Requerido: Sí

4. **Nacionalidad**
   - Tipo: text
   - Ancho: 1 columna
   - Placeholder: "Ej: Peruana"

5. **Fecha de nacimiento**
   - Tipo: date
   - Ancho: 1 columna
   - Requerido: Sí

6. **Edad**
   - Tipo: number (readonly, calculado automáticamente desde fecha de nacimiento)
   - Ancho: 1 columna
   - Nota: Agregar JS para calcular edad automáticamente

7. **DNI**
   - Tipo: text (8 dígitos)
   - Ancho: 1 columna
   - Placeholder: "12345678"

8. **Carné de Extranjería**
   - Tipo: text
   - Ancho: 1 columna
   - Placeholder: "Si aplica"

   - Tipo: select dropdown
   - Opciones:
     - Sin educación
     - Primaria incompleta
     - Primaria completa
     - Secundaria incompleta
     - Secundaria completa
     - Técnica
     - Universitaria incompleta
     - Universitaria completa
   - Ancho: 1 columna

10. **Estado Civil**
    - Tipo: radio buttons (5 opciones en línea)
    - Opciones: Soltero(a) | Conviviente | Casado(a) | Separado(a) | Viudo(a)
    - Ancho: 2-3 columnas (span completo)

11. **N° de hijos**
    - Tipo: number
    - Ancho: 1 columna
    - Min: 0

12. **Domicilio actual**
    - Tipo: text
    - Ancho: 2 columnas (más largo)
    - Placeholder: "Calle, Av., Jr., etc."

13. **Distrito**
    - Tipo: text
    - Ancho: 1 columna
    - Placeholder: "Ej: Chiclayo"
```
